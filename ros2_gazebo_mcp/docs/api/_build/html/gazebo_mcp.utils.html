gazebo_mcp.utils.converters module

Message Converters for Gazebo MCP.

Provides conversion functions between ROS2 message types and Python dictionaries: - Pose, Twist, Transform conversions - Quaternion ↔ Euler angle conversions - JSON serialization for ROS2 messages

These converters enable agent-friendly data formats while maintaining compatibility with ROS2 interfaces.

gazebo_mcp.utils.converters.quaternion_to_euler(x: float, y: float, z: float, w: float) → Tuple[float, float, float][source]

Convert quaternion to Euler angles (roll, pitch, yaw).

Uses the ZYX rotation sequence (intrinsic rotations).

Parameters:

  • x – Quaternion components

  • y – Quaternion components

  • z – Quaternion components

  • w – Quaternion components

Returns:

Tuple of (roll, pitch, yaw) in radians

Example

>>> # Identity quaternion (no rotation):
>>> roll, pitch, yaw = quaternion_to_euler(0.0, 0.0, 0.0, 1.0)
>>> # Returns: (0.0, 0.0, 0.0)
>>>
>>> # 90-degree rotation around Z-axis:
>>> roll, pitch, yaw = quaternion_to_euler(0.0, 0.0, 0.707, 0.707)
>>> # Returns: (0.0, 0.0, 1.57)  # ~π/2 radians

gazebo_mcp.utils.geometry module

Geometry Utilities for Gazebo MCP.

Provides geometric operations for robotics: - Quaternion operations (multiply, conjugate, inverse, normalize) - Euler ↔ Quaternion conversions - Transform composition and inversion - Distance and angle calculations - Coordinate frame transformations

All angles are in radians unless specified otherwise.

gazebo_mcp.utils.geometry.quaternion_multiply(q1: Tuple[float, float, float, float], q2: Tuple[float, float, float, float]) → Tuple[float, float, float, float][source]

Multiply two quaternions.

Quaternion multiplication is used to compose rotations. Order matters: q1 * q2 ≠ q2 * q1

Parameters:

  • q1 – First quaternion (x, y, z, w)

  • q2 – Second quaternion (x, y, z, w)

Returns:

Result quaternion (x, y, z, w)

Example

>>> # Rotate 90° around Z, then 90° around X:
>>> q_z = euler_to_quaternion(0, 0, math.pi/2)  # 90° around Z
>>> q_x = euler_to_quaternion(math.pi/2, 0, 0)  # 90° around X
>>> q_result = quaternion_multiply(q_z, q_x)
